# טעויות נפוצות של מטפלים

## 1. פרשנות מהירה
המטפל “מבין” מהר מדי → מפספס את האדם.

## 2. הובלה יתר
המטפל דוחף תהליך במקום לעקוב אחרי המטופל.

## 3. דרמטיזציה
יצירת דרמה במקום דיוק.

## 4. תפקיד הוריי
המטפל מחזיק את המטופל במקום לאפשר לו להחזיק עצמו.

## 5. ייעוץ במקום חקירה
המטפל נותן פתרונות במקום לאפשר גילוי.
