# מה מחבל בהקשבה

## גורמים מרכזיים
1. שיפוטיות.  
2. אגו שמבקש "להיות צודק".  
3. פחד לאבד שליטה.  
4. רעש פנימי.  
5. פרשנות מהירה מדי.  

## שיטה להתגבר
נשימה → סקרנות → שאלת עומק → מרחב פנימי.
