# אנחנו נשאל בכל זאת אז מה צריך להשתנות ככה ביחס הפנימי כדי שתוכלי להחזיר טיפה את

## נקודות עומק:

- אנחנו נשאל בכל זאת אז מה צריך להשתנות ככה ביחס הפנימי כדי שתוכלי להחזיר טיפה את הנוכחות הזו שלא יכולת להביא אותה כילדה ואת מתקשה להביא אותה כאם

- חייבת כן תמשיכי הנה את מתקדמת

## תנועה/יישום:

- יש לזה שם כל הדבר הזה אומר להחזיר את הילדיות את רואה את החיוך הזה ראיתם איזה חיוך

- איזה כיף שיש שם זה מה ש
